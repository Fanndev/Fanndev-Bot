
# FANNDEV-BOT

Gabut sebagai latihan aja


## Run Locally

Clone the project

```bash
  git clone https://github.com/Fanndev/Fanndev-Bot.git
```

Go to the project directory

```bash
  cd Fanndev-Bot
```

Install dependencies

```bash
  npm install / npm i
```

Start the server

```bash
  npm run dev
```
